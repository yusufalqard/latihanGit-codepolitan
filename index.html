Git Clone lalu masukkan link dari repo seperti contoh https://github.com/yusufalqard/hawi-react-portofolio

Git add untuk menambah file secara terpisah atau keseluruhan

Git commit dapat digunakan sebagai pesan perubahan code yang dikerjakan
